## 🌱 Auto Plant Watering System

This project is an automated plant watering system designed to monitor soil moisture levels and water plants only when necessary. The system reduces manual effort while ensuring efficient water usage and healthier plant maintenance.

---

### 🔧 How It Works

- A soil moisture sensor continuously measures moisture levels  
- When moisture drops below a defined threshold, the system activates a water pump  
- Once adequate moisture is restored, the system stops watering  

This allows the plant to maintain optimal conditions without constant human intervention.

---

### 💡 Key Features

- Automated watering based on real-time sensor data  
- Efficient water usage  
- Simple and reliable control logic  
- Reduces risk of overwatering or underwatering
